Explain Constructor Chaining with an example?
Ans:- Constructor chaining is a technique in object-oriented programming 
where a constructor in a class calls another constructor in the same
 class (or in the parent class)
This is useful for reusing code and reducing redundancy.

Explain Inheritance with an Example?

(the parent or superclass) it is  relationship between classes.

Explain Encapsulation with an Example?

Ans:-Encapsulation is (attributes) and methods (functions) 
that operate on the data into a single unit or class. 
It also involves restricting direct access to some of the object
 which helps to protect the internal state of the object
